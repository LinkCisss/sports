# 更新日志 (Update Log) - 2026年5月22日

在今天的更新中，我们对整个 App 进行了深度重构与功能迭代，使其达到了“豪华级别”的视觉质感，并成功跑通了与后端云数据库（Supabase）的完整链路交互。

## 💎 1. 极致黑金美学 UI (Deep Luxury UI)
- **极黑深色模式**：摒弃了普通的灰色背景，采用了真正的 OLED 黑（`#0F0F0F`）作为页面背景色，搭配 `#1C1C1E` 的高级磨砂黑作为卡片背景，对比度极强。
- **高定排版与交互**：
  - 提高了球队名称字体大小（大字重 `22-28px`），提升视觉层级。
  - 所有包含赔率的数字采用了等宽数字（`tabular-nums`），实现了不同行之间的严格纵向对齐。
  - `MatchCard` 的圆角升级至夸张而高级的 `24px`，并且加入了物理级别的下按缩放交互效果 (`scale: 0.98`)。
- **动态呼吸阴影**：在浅色和深色模式下，为主控按钮和卡片添加了带有强调色 (`#FF3B5C`) 光晕的彩色弥散投影。

## 🗺️ 2. 全球国旗集成引擎
- 集成了 `react-native-country-flag`，用真实且高清的国旗取代了原本乏味的灰色占位符。
- 建立并完善了底层 `utils/flags.ts` 映射字典，完美涵盖了：
  - **欧洲主流俱乐部**（如皇马、曼城、拜仁等）。
  - **2026 世界杯主要国家队**（南美双雄、欧洲豪门、亚洲诸强、非洲雄鹰等数十支队伍）。
- **智能降级显示**：当遇到未收录的冷门球队时，应用会自动隐藏占位符并将球队名字左对齐，保证卡片永远干净美观。

## 📅 3. 动态时间轴与世界杯特调逻辑
- 首页顶部的日期选择器被赋予了智能逻辑：当用户选择 **World Cup (世界杯)** 时，日期滑块会瞬间跳转锁定在 **2026年6月11日**，并动态渲染后续的 14 个比赛日。
- 如果当天暂无比赛数据，屏幕居中会展示极具亲和力的浅灰色友好提示，彻底消除了空界面的突兀感。

## 🚀 4. Apple Liquid Glass (毛玻璃底部导航栏)
- 废弃了传统的白色底栏，重构了 `app/(tabs)/_layout.tsx`。
- 启用了 `expo-blur` 组件，并且将 `intensity` 参数拉满至 `100`，配合绝对定位，在用户滑动比赛卡片时，底部 Tab 会产生极为迷幻和高端的 iOS 原生毛玻璃透视效果。

## ❤️ 5. Supabase 匿名收藏系统
- 移除了原先的“预测”页面，改组为 **「Favorites / 我的收藏」** 页面。
- 编写了 `lib/favorites.ts` 数据库交互层：
  - **无痛使用体验**：使用原生 `AsyncStorage` 配合 UUID v4 生成器给每台设备下发随机指纹，利用 `user_id` 实现了“免登录直接收藏”。
  - **红心点亮逻辑**：在 `MatchCard` 头部的右侧植入了一颗空心红心。点击时它会阻断 React Native 的事件冒泡（防止误跳详情页），然后瞬间点亮并把 **整场比赛数据 (`match_data: JSONB`)** 直接丢进 Supabase。这样即使用户收藏的比赛在 API 里过期消失，依然能在“收藏页”中被完美渲染！

## 🔗 6. 高转化 Affiliate 导流入口
- 在比赛详情页（`app/match/[id].tsx`）的底部，悬浮了一个巨大的、具有极强点击欲的深红色「去 1xBet 投注」操作胶囊 (Floating Action Button)。
- 当用户挑选完各个平台的赔率之后，可以点击该按钮直接通过 `expo-web-browser` 无缝唤出你的专属推广链接：`https://reffpa.com/L?tag=d_5622403...`，极大地提高了潜在的变现转化率。
